Identifying Diabetic Retinopathy with Computer Vision techniques

November 2023 - Present

Under team members representing MIGHTE and Professor Mauricio Santillana's guidance, I am working to create a tool that can identify if diabetic retinopathy exists within a vectorized image of a patients eyes. By vectorizing the data and using smaller storage space per image, the hope is to create an identifying computer vision tool that is accessible to all.

Case Study on New London County Drug Overdoses

January 2022 - May 2022

In collaboration with Jennifer Muggeo and Ledge Light Health District, this project aims to create informative and interactive visualizations displaying drug overdose trends in the New London county area of Connecticut. The goal of this project is to communicate important insights into drug overdoses when specifically looking at factors such as race, location, and the drugs involved. These observations assisted the health department in creating informative community events with the goal of preventing future overdoses.


Image Editor GUI

November 2021 - December 2021

As part of a final class project for Object-Oriented Design, I created an image editor GUI. This editor was built with the capability of applying filters, mosaicking, support for script commands, color transformations, blurring, and sharpening on an image within the users computer. The fully functional GUI interface incorperated object-oriented practices.
